\newcounter{commentrow}

\newenvironment{reviewertable}[1]{%
  \setcounter{commentrow}{0}%
  \small
  \begin{longtable}{@{} I R A @{}}
  \toprule
  \rowcolor{headerblue}
  \textbf{ID} & \textbf{Reviewer comment} & \textbf{Author response and action}\\
  \midrule
  \endfirsthead

  \toprule
  \rowcolor{headerblue}
  \textbf{ID} & \textbf{Reviewer comment} & \textbf{Author response and action}\\
  \midrule
  \endhead

  \midrule
  \multicolumn{3}{r}{\emph{Continued on next page}}\\
  \endfoot

  \bottomrule
  \endlastfoot
}{%
  \end{longtable}
}

\newenvironment{reviewersection}[1]{%
  \section*{#1}%
  \begin{reviewertable}{#1}%
}{%
  \end{reviewertable}%
}

% FIXED: Section header row uses a defined-width paragraph box spanning 100% of \linewidth
\newcommand{\SectionRow}[1]{%
  \rowcolor{sectionblue}%
  \multicolumn{3}{@{}L{\linewidth}@{}}{\large\bfseries \hspace{5pt}#1}\\
  \hline
}

% Syntax: \CommentRow{ID}{Label}{Comment}{Response}{ActionText}
\newcommand{\CommentRow}[5]{%
  \noalign{%
    \stepcounter{commentrow}%
    \ifodd\value{commentrow}%
      \global\let\currentrowcolor\relax
    \else
      \gdef\currentrowcolor{\rowcolor{rowgray}}%
    \fi
  }%
  \currentrowcolor
  \phantomsection\label{#2}#1 & #3 & #4%
  % If the 5th argument is not completely blank, format and print it
  \if\relax\detokenize{#5}\relax
    % If argument #5 is empty, do nothing
  \else
    \par\vspace{0.4em}\textbf{Action:} #5%
  \fi
  \\ \hline
}

We thank the reviewers for their constructive comments and suggestions that helped
improve the quality of the manuscript. The manuscript has undergone a revision according to the
reviewers' comments. Please see below our response.

We group comments by reviewer and by their review sections.
Each comment is assigned an ID (for example, \texttt{R1.C1}), which can be referenced later.

}{}
